Bike Sales Analysis Dashboard
Overview

This project demonstrates my ability to execute an end-to-end data analysis process using Excel. My goal was to analyze customer demographic data to uncover the key factors influencing bike purchases.
Tools Used

    Excel: Data cleaning, Pivot Tables, Pivot Charts, Slicers, and Dashboard design.

Project Highlights

    Data Cleaning: Removed duplicates and standardized categorical labels (e.g., converting 'M' to 'Married').
    Feature Engineering: Categorized age data into 'Adolescent', 'Middle Age', and 'Old' for clearer trend analysis.
    Interactive Dashboard: Created a centralized view with slicers, enabling users to filter by Marital Status, Education, and Region.

Key Insights

    Identified clear correlations between income levels and the likelihood of purchasing a bike.
    Discovered which specific age brackets show the highest engagement in bike buying.

How to Use

    Download the Excel file from this repository.
    Open the file in Excel.
    Navigate to the 'Dashboard' sheet and use the slicers to filter the data interactively.
